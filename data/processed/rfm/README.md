## RFM Dataset

This folder contains the final customer-level RFM dataset used for segmentation
and dashboard analysis.

File description:
- rfm_final_profile.xlsx  
  Contains Recency, Frequency, and Monetary metrics aggregated at customer level,
  along with RFM scores and segment labels.

This dataset represents the final analytical output and is included to support:
- Transparency of customer segmentation
- Validation of RFM logic
- Dashboard development
